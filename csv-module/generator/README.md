# How to Use

1. config/csv-mapper.js 수정
   > 원하는 데이터의 headers와 generatatorMandatoryMap 값으로 셋팅

2. config/csv-config.json 수정 
   > path : 생성할 csv의 명칭,
   > rows : 생성할 row 갯수  


# Run
node index.js
